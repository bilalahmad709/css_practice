PRACTICE OF DISPLAY PROPERTY

In CSS, "display" is a property that determines how an HTML element is visually rendered on a web page. It controls the box type generated for the element and how it interacts with other elements in the layout.


DISPLAY:INLINE :-

Elements only take up the necessary width and do not cause a line break. They flow alongside each other horizontally.
e.g:

Lorem ipsum dolor sit amet, consectetur adipisicing elit. Ipsa, totam?


DISPLAY:NONE :-

Elements are not displayed, effectively making them invisible and not taking up any space.
